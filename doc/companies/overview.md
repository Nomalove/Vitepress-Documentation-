# Companies on Wrapped
 
Production companies use Wrapped to:

- Post and manage job listings.

- Browse verified freelancer profiles.

- Manage budgets, documents, and schedules.

- Review and rate freelancers after each project.
 
The production dashboard is about 90% complete and integrates:

- Job posting

- Calendar scheduling

- Linked profiles (crew, cast, etc.)
